# if and choose statements - JSLT

Jslt If statement works as Java does. 

>  To test something we  write the following code:

```jsp
<c:if test="${Math.random()*11 > 5}">
    <h1>Success!</h1>
</c:if>
```

> To test several cases we can use:

```jsp
 <c:choose>
        <c:when test='${5 > 3}'>
            <h1> Success! ;)</h1>
        </c:when>
        <c:otherwise>
            <h1>Fail :( </h1>    
         </c:otherwise>
</c:choose>
```

**That's it, simple.**
