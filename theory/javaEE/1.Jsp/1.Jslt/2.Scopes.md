# JSLT scopes

JSLT also works with scopes. 

## Scopes

- Page

- Request

- Session

- Application

## To use them

Getting them: 

```jsp
<input name="date" value="${pageScope.today}">
```

Setting them

```jsp
<c:set var="names" value="A B C, D" scope="page" />
<!--OR YOU CAN USE-->
<%
    Char[] names = new char[]{'A','B','C'}
    pageContext.setAttribute("names", names);
%>
```
